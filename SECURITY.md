## Reporting a Vulnerability

We take the security of OGT WatchTower seriously. If you have found a security vulnerability in the application, please DO NOT report it on the public GitHub issue tracker.

### How to Report

Please email us at **security@ogt-security.com** with the details of the vulnerability. We will verify the issue and get back to you within 48 hours.

### What to Include

- Step-by-step instructions to reproduce the issue
- Affected version of OGT WatchTower
- Potential impact of the vulnerability

We appreciate your responsible disclosure and will credit you in our release notes for any verified security issues.
